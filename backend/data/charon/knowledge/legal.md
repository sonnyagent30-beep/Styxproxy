## What data we collect

For website orders: almost nothing. The only identifier is a small file stored on the customer's own device that links their purchase history to their browser. Clearing site data removes the link.

For website chat (Charon): messages sent during the conversation (retained 90 days). Anonymous session behaviour data — pages visited, time on pages, scroll depth, cart actions — stored in the customer's browser only and used only in aggregate to improve Charon's proactive support. No personal information is associated with this data.

For Telegram and WhatsApp orders: the chat ID, messages, and any delivery information the customer provides. Customers can request deletion of this data at any time by sending "delete my data" to the Telegram bot or emailing privacy@styxproxy.com.

## What we never do

- Never store cookies on customer browsers.
- Never use third-party tracking pixels or analytics that fingerprint the visitor.
- Never link session behaviour data to any individual customer.
- Never share customer data with anyone except a payment processor necessary to process the order, and law enforcement only when legally compelled.

## Acceptable use

The proxy cannot be used for:

- Anything illegal under the customer's jurisdiction or ours.
- Email spam, phishing, or mass unsolicited messaging.
- Hacking, fraud, or unauthorized access to systems.
- Child exploitation material.
- Anything that violates another party's terms of service in a way that puts Styxproxy at legal risk.

The full list is on the Acceptable Use page.

## Payment processor

All payments are processed by Flutterwave. Styxproxy never sees the customer's card number or full bank details, only the transaction reference and payment status.

For refund requests, the customer should email support@styxproxy.com with their transaction reference. Refund requests are reviewed by a human.

## Order retrieval and proxy recovery

If a customer paid but did not receive their proxy credentials:

1. Chat with Charon on the website — say "I paid but didn't get my proxy" and share the Flutterwave transaction reference when asked. Charon can look up the order and deliver credentials directly in the chat.
2. Go to styxproxy.com/manage and enter the transaction reference.
3. Message @styxproxy on Telegram with the transaction reference.
4. Email support@styxproxy.com with the transaction reference.

The transaction reference is the customer's only order identifier for website orders. Charon can look up any order by this reference and retrieve credentials.
