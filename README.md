### Instant (Website) вҖ” Primary Path
- Select product вҶ’ pay via Flutterwave вҶ’ IP displayed on screen
- **No registration. No email required. No data collected.**
- Order number (tx_ref) is the only identifier

### Telegram + WhatsApp вҖ” Full Ordering + Support
- Order via chat вҶ’ payment link вҶ’ IP delivered in chat
- Free trial (chat only вҖ” Theorem Reach requires a postback URL)
- Full support: check status, renew, ban claims, questions

### Management Portal
- `styxproxy.com/manage` вҶ’ enter tx_ref (no login)
- Check status, renew, report ban вҶ’ redirects to Telegram or WhatsApp

## Styxproxy Auth Layer

Customers receive Styxproxy-branded credentials. Actual proxy IPs are sourced from vetted infrastructure partners but customers interact only with Styxproxy.

```
Customer sees:   proxy1.styxproxy.com:1080
Behind the scenes: actual provider IP вҶ’ routed through Styxproxy auth
```

This enables instant revoke on refund and free trial recycling.
